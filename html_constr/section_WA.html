Słówka

Język: Python 3.5

Framework: Qt5

Typ: Aplikacja Okienkowa / Konsola

Wersja językowa: POL/ANG

Opis: Aplikacja okienkowa do nauki słówek języka Angielskiego. Program sprawdza tłumaczenie słówek POL - ANG i ANG - POL. Po utworzeniu użytkownika tłumaczone słówka przechowywane są w bazie danych. Użytkownik może przetestować swoją pamięć i odpytać się z przechowywanych (sprawdzanych) słówek w wersji POL - ANG i ANG - POL. Weryfikacja poprawności wpisywanych słówek odbywa się, aż do otrzymania poprawnego wyniku. W przypadku podania złego słowa poprawnie wpisane litery są zapamiętywane. Program ponownie pyta o słowo z odgadniętymi literami.

Pogoda

Język: Python 3.5

Typ: Applikacja konsolawa

Wersja językowa: POL

Opis: Program do pobierania i przetwarzania danych pogodowych. Aplikacja pobiera aktualne dane pogodowe (w formacie XML) z najbliższej stacji meteorologicznej i zapisuje do piliku typu CSV. Dane archiwalne pobierane są przez program ze stron internetowych (w formacie XML) . Program sortuje pozyskane dane, a następnie scala je i zapisuje to wspólnego pliku CSV. Dane można obejrzeć z poziomu konsoli lub pliku. Dostępne możliwości to codzienna temperatura i opad oraz średnia arytmetyczna dla wybranego okresu temperatury lub opadu. Uruchomiony program pobiera aktualne dane co 15min.

Link:
